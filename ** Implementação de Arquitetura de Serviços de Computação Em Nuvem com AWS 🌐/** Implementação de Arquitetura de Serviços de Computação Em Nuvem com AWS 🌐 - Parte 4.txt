As aplicações em nuvem são as únicas que são executadas diretamente na nuvem. Elas podem ser criadas e executadas em diferentes regiões do mundo, o que permite às organizações criar e executar aplicações em nuvem de forma escalável e flexível. A AWS oferece uma ampla gama de serviços para criar e executar aplicações em nuvem, incluindo serviços como API Gateway, Lambda e S3.

▸ Principais serviços para aplicações em nuvem da AWS:
  → API Gateway: gerenciamento de APIs
  → Lambda: execução de funções serverless
  → S3: armazenamento de dados para aplicações

→ As aplicações em nuvem são essenciais para criar e executar aplicações em nuvem.
