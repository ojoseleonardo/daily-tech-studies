Ansible se integra facilmente a pipelines de CI/CD como GitHub Actions, GitLab CI ou Jenkins. Um exemplo simples no GitHub Actions:

```yaml
name: Deploy
on: push
jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Run Ansible Playbook
        run: |
          ansible-playbook -i inventory.yml site.yml --limit production --vault-password-file .vault_pass
```
