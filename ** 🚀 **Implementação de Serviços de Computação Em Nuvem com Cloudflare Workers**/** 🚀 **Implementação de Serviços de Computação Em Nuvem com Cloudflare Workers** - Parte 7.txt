▸ Principais maneiras de gerenciamento:
  → Ferramentas de armazenamento: utilizar ferramentas de armazenamento para armazenar os dados
  → Ferramentas de gerenciamento de dados: utilizar ferramentas de gerenciamento de dados para gerenciar os dados
  → Práticas de gerenciamento de dados: utilizar práticas de gerenciamento de dados para garantir a integridade dos dados
