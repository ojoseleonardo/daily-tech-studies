A orquestração de containers é uma abordagem que permite a gerenciamento de containers em larga escala. Isso permite a criação de ambientes escaláveis e confiáveis.

▸ Principais benefícios:
  → Ambientes escaláveis e confiáveis
  → Maior flexibilidade para mudanças no mercado
  → Menor tempo de implementação

→ Exemplo de uso da orquestração: Kubernetes
